# Convertisseur de bases (2, 8, 10, 16)

Application web simple en **HTML, CSS et JavaScript** permettant de convertir des nombres entre les bases **binaire (2)**, **octale (8)**, **décimale (10)** et **hexadécimale (16)**.

## Fonctionnalités
- Conversion entre les bases 2, 8, 10 et 16
- Interface simple et responsive
- Exécution 100 % côté client

## Arborescence du projet

/project

├─ js/

## Technologies utilisées
- HTML5
- CSS3
- JavaScript (Vanilla)

## Utilisation
1. Cloner ou télécharger le projet
2. Ouvrir `index.html` dans un navigateur web
3. Saisir un nombre et sélectionner les bases souhaitées
